# 🚀 Instalación Rápida - Simpsons Online Pro

## ✅ Instalación en 5 Pasos (5 minutos)

### Paso 1: Subir el Tema (2 min)

1. Comprimir carpeta `simpsons-online-theme` en ZIP
2. WordPress → **Apariencia** → **Temas** → **Añadir nuevo**
3. **Subir tema** → Seleccionar ZIP
4. **Instalar ahora** → **Activar**

### Paso 2: Instalar ACF (1 min)

1. **Plugins** → **Añadir nuevo**
2. Buscar: "Advanced Custom Fields"
3. Instalar y activar

### Paso 3: Configurar Campos ACF (1 min)

Crear 3 campos tipo "Área de texto":
- Nombre: `player1` (obligatorio)
- Nombre: `player2` (opcional)
- Nombre: `player3` (opcional)

Ubicación: Tipo de entrada = Entrada

### Paso 4: Crear Categorías (30 seg)

Crear temporadas:
- Temporada 1
- Temporada 2
- Temporada 3
- ... (hasta tu temporada actual)
- Especiales
- La Película

### Paso 5: Configurar Menú (30 seg)

1. **Apariencia** → **Menús**
2. Crear menú con temporadas
3. Asignar a "Menú Principal"
